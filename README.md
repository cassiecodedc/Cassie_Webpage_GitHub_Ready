# Cassie Kexin Huang Portfolio

Static personal portfolio website for GitHub Pages or any basic web host.

## Pages

- `index.html` - homepage and profile overview
- `journalism.html` - reporting, research, and independent analysis
- `visual-communication.html` - video, photography, podcast, and immersive work
- `data-storytelling.html` - visualization, consulting, and survey research projects
- `design.html` - editorial design, zine, product, and documentary projects

## Deploying to GitHub Pages

Upload the files in this folder to a GitHub repository, then enable GitHub Pages from the repository settings. Use `index.html` as the homepage.
